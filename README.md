# ClassTrack Attendance App

This is your college attendance management system.

## How to Resume Work

1. **Open this folder in Antigravity**: Use the "Open Folder" button in your UI and select `C:\Users\aju85\.gemini\antigravity\scratch\attendance_app`.
2. **Start the server**: Double-click the `run.bat` file in this directory.
3. **View the website**: Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure
- `index.html`: Login page (Entry point)
- `admin.html`: Administrator dashboard
- `teacher.html`: Teacher portal
- `student.html`: Student portal
- `styles.css`: Main styling file
- `app.js`: Application logic
